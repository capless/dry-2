        console.log('Slide changed:', event.detail);

API Reference

Carousel Properties

Property Type Default Description
autoplay boolean false Whether slides should advance automatically
interval number 5000 Time in milliseconds between slide transitions
pause-on-hover boolean false Whether autoplay should pause when hovering
navigation boolean false Whether to show previous/next buttons
indicators boolean false Whether to show indicator dots
class string '' Additional CSS classes for the carousel container
button-class string '' Additional CSS classes for navigation buttons
indicator-class string '' Additional CSS classes for indicators container

Carousel Methods

Method Description
prevSlide() Goes to the previous slide
nextSlide() Goes to the next slide
goToSlide(index) Goes to the slide at the specified index
pauseAutoplay() Pauses autoplay if it's enabled
resumeAutoplay() Resumes autoplay if it was paused

Carousel Events

Event Description
carousel:slide Fired when a slide changes, with detail containing carousel reference, current slide index, and previous slide index
