Web Components Examples

This page demonstrates the usage of four web components:

  • ajax-dialog - A dialog component that loads content via AJAX
  • date-picker - A calendar-based date picker component
  • toggle-switch - A toggle switch for boolean inputs
  • super-select - An enhanced select dropdown with search functionality

AJAX Dialog

The AJAX Dialog component creates a modal dialog that loads content dynamically via AJAX using HTMX.

Test Dialog
<ajax-dialog url="dialog-content.html" button-class="bg-blue-500 hover:bg-blue-600 text-white px-4 py-2 rounded" dialog-class="p-6 max-w-md rounded-lg" dialog-inner-id="demo-dialog-content"> Open Dialog </ajax-dialog>

Attributes

Attribute Description Default
url URL to fetch content from ""
button-class CSS classes for the trigger button ""
dialog-class CSS classes for the dialog element "bg-white"
dialog-inner-id ID for the content container "dialog-inner"
trigger-id ID for the trigger element "trigger"
trigger-type HTMX trigger type "click"

JavaScript Usage

You can also interact with these components programmatically:

// Get a reference to a date picker const datePicker = document.querySelector('date-picker'); // Set a date programmatically datePicker.selectedDate = new Date(2025, 2, 30); // March 30, 2025 datePicker.input.value = datePicker.formatDate(datePicker.selectedDate); datePicker.updateCalendarView(); // Listen for changes datePicker.addEventListener('change', (event) => { console.log('Selected date:', datePicker.selectedDate); }); // Get a reference to a super-select const superSelect = document.querySelector('super-select'); // Set value programmatically superSelect.value = 'option2'; // For single select // or superSelect.value = ['option1', 'option3']; // For multiple select // Get the current value console.log(superSelect.value); // Toggle switch state const toggle = document.querySelector('toggle-switch'); const input = toggle.querySelector('input[type="checkbox"]'); input.checked = true; // Programmatically check the toggle

JavaScript Usage

// Get a reference to the tabs component const tabs = document.querySelector('tabs-component'); // Change the active tab programmatically (zero-based index) tabs.activeTab = 1; // Switch to the second tab // Listen for tab change events tabs.addEventListener('tab-changed', (event) => { console.log('Active tab changed to:', event.detail.index); // Do something when the tab changes });

Toast Component

The Toast component provides temporary notifications that appear and disappear automatically.

Basic Toast Types

Toast Positions

Custom Duration

Multiple Toasts

<!-- Basic toast created declaratively --> <toast-component type="success" position="bottom-right" duration="3000" auto-show> Your changes have been saved! </toast-component> <!-- Creating toasts with JavaScript --> <script> // Show a simple info toast Toast.info('This is an informational message.'); // Show a success toast with custom position Toast.success('Operation completed successfully!', { position: 'top-right' }); // Show a warning toast with custom duration Toast.warning('Please review your input.', { duration: 5000 }); // Show an error toast with custom position and duration Toast.error('An error occurred.', { position: 'top-center', duration: 8000 }); // Show a persistent toast (stays until dismissed) Toast.info('This toast stays until dismissed.', { duration: 0 }); </script>

Attributes

Attribute Description Default
type Toast type: info, success, warning, error "info"
position Toast position on screen "bottom-right"
duration Time in milliseconds before auto-dismiss (0 for persistent) 3000
message Toast message (alternative to innerHTML) ""
auto-show Show toast automatically when created false
toast-class Custom CSS classes for the toast element Based on type
container-class Custom CSS classes for the toast container Based on position

Static Methods

Method Description
Toast.info(message, options) Show an info toast
Toast.success(message, options) Show a success toast
Toast.warning(message, options) Show a warning toast
Toast.error(message, options) Show an error toast
Toast.showToast(message, options) Generic method to show a toast with custom options

Events

Event Description
speed-dial:opened Fired when the speed dial is opened
speed-dial:closed Fired when the speed dial is closed
speed-dial:action Fired when an action item is clicked, with item details in event.detail

JavaScript API

// Get a reference to the speed dial
const speedDial = document.querySelector('speed-dial');

// Open the speed dial
speedDial.open();

// Close the speed dial
speedDial.close();

// Toggle the speed dial
speedDial.toggle();

// Listen for events
speedDial.addEventListener('speed-dial:action', (e) => {
  console.log('Action clicked:', e.detail.label);
  // e.detail contains index, action, and label
});
