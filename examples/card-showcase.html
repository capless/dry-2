Card Component

A flexible container for displaying related content with consistent styling.

Basic Card

A simple card with default styling.

This is a basic card with default styling. The content is placed in the card body automatically.

 
<card-component>
    <p>This is a basic card with default styling. The content is placed in the card body automatically.</p>
</card-component>

Card with Sections

A card with header, body, and footer sections.

Card Title

This is the main content of the card. You can place any content here including text, images, and other components.

 
<card-component>
    <div slot="header">
        <h3 class="text-lg font-semibold">Card Title</h3>
    </div>
    <div slot="body">
        <p>This is the main content of the card. You can place any content here including text, images, and other components.</p>
    </div>
    <div slot="footer">
        <div class="flex justify-end">
            <button class="px-4 py-2 bg-blue-500 text-white rounded">Action</button>
        </div>
    </div>
</card-component>

Card with Media

A card that includes media content.

Media Card

This card includes an image at the top. You can also use the horizontal layout to place the image to the side.

 
<card-component>
    <div slot="media">
        <img src="https://via.placeholder.com/800x400" alt="Placeholder" class="w-full h-48 object-cover">
    </div>
    <div slot="body">
        <h3 class="text-lg font-semibold mb-2">Media Card</h3>
        <p>This card includes an image at the top. You can also use the horizontal layout to place the image to the side.</p>
    </div>
</card-component>

Card Layouts

Cards can be displayed in vertical or horizontal layouts.

Horizontal Layout

Horizontal Card

This card uses a horizontal layout with the media on the left and content on the right.

 
<card-component layout="horizontal">
    <div slot="media">
        <img src="https://via.placeholder.com/300x300" alt="Placeholder" class="w-32 h-full object-cover">
    </div>
    <div slot="body">
        <h3 class="text-lg font-semibold mb-2">Horizontal Card</h3>
        <p>This card uses a horizontal layout with the media on the left and content on the right.</p>
    </div>
</card-component>

Interactive Card

Cards can be made interactive with hover and focus states.

Interactive Card

This card is interactive. Click or use keyboard to trigger events.

Click me!

Event Output:

No events yet. Click the card above.
<card-component interactive>
    <div slot="body">
        <h3 class="font-medium mb-2">Interactive Card</h3>
        <p>This card is interactive. Click or use keyboard to trigger events.</p>
        <p class="mt-2 text-blue-500">Click me!</p>
    </div>
</card-component>

<script>
    document.addEventListener('card:click', (event) => {
        console.log('Card clicked:', event.detail);
        document.getElementById('event-output').textContent = 
            `Card clicked at ${new Date().toLocaleTimeString()}`;
    });
</script>

API Reference

Card Properties

Property Type Default Description
elevation string 'md' Shadow size: 'none', 'sm', 'md', 'lg', 'xl'
variant string 'default' Visual style: 'default', 'outlined', 'filled'
interactive boolean false Whether the card is interactive (clickable)
layout string 'vertical' Card layout: 'vertical', 'horizontal'
header-class string 'p-4 border-b border-gray-200 font-medium' CSS classes for header section
media-class string '' CSS classes for media section
body-class string 'p-4' CSS classes for body section
footer-class string 'p-4 border-t border-gray-200 bg-gray-50' CSS classes for footer section

Card Slots

Slot Name Description
header Card header content
media Card media content (images, videos)
body Card main content
footer Card footer content
(default) Fallback slot (goes to body)

Card Events

Event Description
card:click Fired when an interactive card is clicked or activated via keyboard
