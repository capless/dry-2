Badge Component

A customizable badge component to display short status indicators, counts, or labels.

Badge Variants

The Badge component supports different color variants.

Primary Success Danger Warning Info 
<dry-badge variant="primary">Primary</dry-badge>
<dry-badge variant="success">Success</dry-badge>
<dry-badge variant="danger">Danger</dry-badge>
<dry-badge variant="warning">Warning</dry-badge>
<dry-badge variant="info">Info</dry-badge>

Badge Sizes

The Badge component supports different sizes.

Small Medium Large 
<dry-badge size="sm">Small</dry-badge>
<dry-badge size="md">Medium</dry-badge>
<dry-badge size="lg">Large</dry-badge>

Numeric Badges

Badges can display numbers with maximum value options.

5 42 100 1000 
<dry-badge>5</dry-badge>
<dry-badge>42</dry-badge>
<dry-badge max="99">100</dry-badge>
<dry-badge max="999">1000</dry-badge>

Dot Badges

Display badges as simple dots without content.

<dry-badge dot size="sm" variant="primary"></dry-badge>
<dry-badge dot size="md" variant="success"></dry-badge>
<dry-badge dot size="lg" variant="danger"></dry-badge>

Positioned Badges

Badges can be positioned relative to other elements.

API Reference

Badge Properties

Property Type Default Description
variant string 'primary' Color variant: 'primary', 'success', 'danger', 'warning', 'info'
size string 'md' Size of the badge: 'sm', 'md', 'lg'
position string 'standalone' Position relative to parent: 'standalone', 'top-right', 'top-left', 'bottom-right', 'bottom-left'
dot boolean false Whether to display as a dot without content
max number undefined Maximum value for numeric content (displays as max+)
visible boolean true Whether the badge is visible
class string '' Additional CSS classes to apply

Badge Methods

Method Description
show() Shows the badge
hide() Hides the badge
toggle() Toggles the visibility of the badge
