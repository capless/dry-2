Badge Component

A customizable badge component to display short status indicators, counts, or labels.

Badge Variants

The Badge component supports different color variants.

Primary Success Danger Warning Info 
<badge-component variant="primary">Primary</badge-component>
<badge-component variant="success">Success</badge-component>
<badge-component variant="danger">Danger</badge-component>
<badge-component variant="warning">Warning</badge-component>
<badge-component variant="info">Info</badge-component>

Badge Sizes

The Badge component supports different sizes.

Small Medium Large 
<badge-component size="sm">Small</badge-component>
<badge-component size="md">Medium</badge-component>
<badge-component size="lg">Large</badge-component>

Numeric Badges

Badges can display numbers with maximum value options.

5 42 100 1000 
<badge-component>5</badge-component>
<badge-component>42</badge-component>
<badge-component max="99">100</badge-component>
<badge-component max="999">1000</badge-component>

Dot Badges

Display badges as simple dots without content.

<badge-component dot size="sm" variant="primary"></badge-component>
<badge-component dot size="md" variant="success"></badge-component>
<badge-component dot size="lg" variant="danger"></badge-component>

API Reference

Badge Properties

Property Type Default Description
variant string 'primary' Color variant: 'primary', 'success', 'danger', 'warning', 'info'
size string 'md' Size of the badge: 'sm', 'md', 'lg'
position string 'standalone' Position relative to parent: 'standalone', 'top-right', 'top-left', 'bottom-right', 'bottom-left'
dot boolean false Whether to display as a dot without content
max number undefined Maximum value for numeric content (displays as max+)
visible boolean true Whether the badge is visible
class string '' Additional CSS classes to apply

Badge Methods

Method Description
show() Shows the badge
hide() Hides the badge
toggle() Toggles the visibility of the badge
