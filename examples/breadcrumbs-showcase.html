JavaScript API

Control breadcrumbs programmatically with the built-in API.

Home Products
Breadcrumb Info: 
// Get breadcrumbs instance
const breadcrumbs = document.getElementById('my-breadcrumbs');

// Add new item
breadcrumbs.addItem('New Category', '/products/new');

// Set active item by index
breadcrumbs.setActive(0);

// Change separator style
breadcrumbs.separator = 'slash';

// Get breadcrumb information
console.log(breadcrumbs.itemCount);   // Number of items
console.log(breadcrumbs.activeIndex); // Index of active item
console.log(breadcrumbs.separator);   // Current separator

// Method chaining
breadcrumbs
    .addItem('Electronics', '/electronics')
    .addItem('Phones', '/electronics/phones')
    .setActive(1);

API Reference

Breadcrumbs Component

Property Type Default Description
separator string 'chevron' Separator style: 'chevron', 'slash', 'bullet', 'arrow', or custom text
class string '' Additional CSS classes for the container
itemCount number - Read-only: Number of breadcrumb items
activeIndex number - Read-only: Index of the active item

Breadcrumb Item

Property Type Default Description
href string undefined URL for the breadcrumb link
icon string undefined HTML for CSS class icons (e.g., Font Awesome)
active boolean false Whether this item is active (last item auto-set)
class string '' Additional CSS classes for the item

Methods

Method Parameters Returns Description
addItem() text, href?, active? this Add a new breadcrumb item
setActive() index this Set which item is active by index
