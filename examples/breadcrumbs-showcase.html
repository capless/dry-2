Breadcrumbs Component

A breadcrumbs component to show navigational hierarchy, helping users understand their location in the application structure.

Basic Breadcrumbs

A simple breadcrumbs navigation with default styling.

Home Products Electronics Smartphones 
<breadcrumbs-component>
    <breadcrumb-item href="/">Home</breadcrumb-item>
    <breadcrumb-item href="/products">Products</breadcrumb-item>
    <breadcrumb-item href="/products/electronics">Electronics</breadcrumb-item>
    <breadcrumb-item>Smartphones</breadcrumb-item>
</breadcrumbs-component>

Breadcrumbs with Dropdown

Breadcrumbs with a dropdown menu for showing additional navigation options.

Note: This example uses JavaScript to toggle the dropdown menu.

Home Categories Electronics Smartphones
Clothing Electronics Home & Garden Toys & Games 
<breadcrumbs-component class="dropdown-breadcrumbs">
    <breadcrumb-item href="/">Home</breadcrumb-item>
    
    <breadcrumb-item 
        id="dropdown-trigger" 
        class="relative cursor-pointer">
        <span class="flex items-center">
            Categories
            <svg xmlns="http://www.w3.org/2000/svg" class="h-4 w-4 ml-1" fill="none" viewBox="0 0 24 24" stroke="currentColor">
                <path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M19 9l-7 7-7-7" />
            </svg>
        </span>
    </breadcrumb-item>
    
    <breadcrumb-item href="/products/electronics">Electronics</breadcrumb-item>
    <breadcrumb-item>Smartphones</breadcrumb-item>
</breadcrumbs-component>

<!-- Dropdown menu (hidden by default) -->
<div id="dropdown-menu" class="hidden absolute mt-2 w-48 bg-white rounded-md shadow-lg py-1 z-50 text-sm">
    <a href="/products/clothing" class="block px-4 py-2 text-gray-700 hover:bg-gray-100">Clothing</a>
    <!-- More dropdown items... -->
</div>

<script>
    // Toggle dropdown visibility on click
    document.getElementById('dropdown-trigger').addEventListener('click', function() {
        const dropdown = document.getElementById('dropdown-menu');
        dropdown.classList.toggle('hidden');
    });
    
    // Hide dropdown when clicking outside
    document.addEventListener('click', function(event) {
        if (!event.target.closest('#dropdown-trigger')) {
            document.getElementById('dropdown-menu').classList.add('hidden');
        }
    });
</script>

API Reference

Breadcrumbs Component Properties

Property Type Default Description
separator string 'chevron' Type of separator: 'chevron', 'slash', 'bullet', or custom HTML/text
class string '' Additional CSS classes for the breadcrumbs container

Breadcrumb Item Properties

Property Type Default Description
href string undefined URL for breadcrumb link. If not provided, item is not a link.
icon string undefined HTML string for an icon (SVG recommended)
active boolean false (auto for last item) Whether this item is active/current. Last item is automatically active.
class string '' Additional CSS classes for the breadcrumb item
