Using collapse components to create an accordion interface.

This is the first section of the accordion example.

Collapse components are ideal for creating accordion interfaces.

This is the second section of the accordion example.

You can open multiple sections or configure it to only allow one section open at a time.

This is the third section of the accordion example.

You can add any content inside the collapsible sections, including other components, images, or interactive elements.

 
<div class="border rounded overflow-hidden divide-y">
    <!-- Accordion Item 1 -->
    <div class="accordion-item">
        <button class="accordion-header w-full p-4 text-left font-medium bg-gray-50">
            <span>Section 1: Introduction</span>
            <svg class="h-5 w-5 transform transition-transform accordion-icon">...</svg>
        </button>
        <collapse-component class="accordion-content">
            <div class="p-4">
                <p>This is the first section of the accordion example.</p>
            </div>
        </collapse-component>
    </div>
    
    <!-- Additional accordion items... -->
</div>

<script>
    // Accordion example code
    document.querySelectorAll('.accordion-header').forEach(header => {
        header.addEventListener('click', () => {
            const content = header.nextElementSibling;
            content.toggle();
            
            // Optional: rotate icon
            header.querySelector('.accordion-icon').classList.toggle('rotate-180');
            
            // Optional: for single-open accordion
            // document.querySelectorAll('.accordion-content').forEach(item => {
            //     if (item !== content && item.expanded) {
            //         item.collapse();
            //     }
            // });
        });
    });
</script>

Event Handling

The collapse component emits events during animation.

This collapse demonstrates event handling.

Watch the event log below as you toggle.

Event Log:

No events yet. Toggle the collapse above to see events.
<button id="event-toggle">Toggle with Events</button>

<collapse-component id="event-collapse">
    <div class="p-4 bg-gray-100 rounded">
        <p>This collapse demonstrates event handling.</p>
    </div>
</collapse-component>

<div id="event-log"></div>

<script>
    document.getElementById('event-toggle').addEventListener('click', () => {
        document.getElementById('event-collapse').toggle();
    });
    
    const eventLog = document.getElementById('event-log');
    
    document.addEventListener('collapse:expanding', (e) => {
        eventLog.textContent = `Expanding started at ${new Date().toLocaleTimeString()}`;
    });
    
    document.addEventListener('collapse:expanded', (e) => {
        eventLog.textContent += `\nExpanded completed at ${new Date().toLocaleTimeString()}`;
    });
    
    document.addEventListener('collapse:collapsing', (e) => {
        eventLog.textContent = `Collapsing started at ${new Date().toLocaleTimeString()}`;
    });
    
    document.addEventListener('collapse:collapsed', (e) => {
        eventLog.textContent += `\nCollapse completed at ${new Date().toLocaleTimeString()}`;
    });
</script>

API Reference

Collapse Properties

Property Type Default Description
expanded boolean false Whether the collapse is expanded
duration number 300 Animation duration in milliseconds
class string '' Additional CSS classes

Collapse Methods

Method Description
toggle() Toggles between expanded and collapsed states
expand() Expands the collapse component
collapse() Collapses the collapse component

Collapse Events

Event Description
collapse:expanding Fired when expansion animation begins
collapse:expanded Fired when expansion animation completes
collapse:collapsing Fired when collapse animation begins
collapse:collapsed Fired when collapse animation completes
