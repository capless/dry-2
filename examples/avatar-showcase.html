Avatar Component

A customizable avatar component to display user profile images with fallback options.

Avatar Sizes

The Avatar component supports different sizes.

<avatar-component name="John Doe" size="xs"></avatar-component>
<avatar-component name="John Doe" size="sm"></avatar-component>
<avatar-component name="John Doe" size="md"></avatar-component>
<avatar-component name="John Doe" size="lg"></avatar-component>
<avatar-component name="John Doe" size="xl"></avatar-component>

Avatar Shapes

The Avatar component can be displayed in different shapes.

<avatar-component name="John Doe" shape="circle"></avatar-component>
<avatar-component name="John Doe" shape="square"></avatar-component>
<avatar-component name="John Doe" shape="rounded"></avatar-component>

Avatar with Images

The Avatar component can display images, with fallback to initials.

<avatar-component 
    name="John Doe" 
    image="https://i.pravatar.cc/150?img=1">
</avatar-component>

Avatar Colors

Avatar initials can be displayed with different background colors.

<avatar-component name="John Doe" bg-color="#3B82F6"></avatar-component>

Avatar Group

Group multiple avatars together with overlap.

<avatar-group max="3" more-count="2">
    <avatar-component name="John Doe" image="..."></avatar-component>
    <avatar-component name="Jane Smith" image="..."></avatar-component>
    <avatar-component name="Robert Johnson" bg-color="#F59E0B"></avatar-component>
    <avatar-component name="Emily Davis" bg-color="#EF4444"></avatar-component>
    <avatar-component name="Michael Brown" bg-color="#8B5CF6"></avatar-component>
</avatar-group>

API Reference

Avatar Properties

Property Type Default Description
name string '' User name used to generate initials
image string null URL of the avatar image
alt string 'Avatar' Alt text for the avatar image
size string 'md' Size of the avatar: 'xs', 'sm', 'md', 'lg', 'xl'
shape string 'circle' Shape of the avatar: 'circle', 'square', 'rounded'
bg-color string random Background color for initials

Avatar Group Properties

Property Type Default Description
max number 3 Maximum number of avatars to display
spacing string '-10px' Spacing between avatars
size string 'md' Size for all avatars in the group
