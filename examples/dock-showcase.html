Dock Component

A dock component that displays a collection of shortcuts or actions in a macOS-style dock interface with magnification effects and tooltips.

API Reference

Dock Component Properties

Property Type Default Description
orientation string 'horizontal' Orientation of the dock: 'horizontal' or 'vertical'
magnification string '2' Magnification factor for the hover effect (e.g., '1.5', '2.5')
class string '' Additional CSS classes for the dock container

Dock Item Properties

Property Type Default Description
label string undefined Label for the dock item (used for tooltip if tooltip not provided)
tooltip string undefined Custom tooltip text (overrides label for tooltip)
href string undefined URL if the dock item is a link
class string '' Additional CSS classes for the dock item

Dock Item Events

Event Description
dock-item:click Fired when a dock item is clicked
