Chat Bubble Component

A chat bubble component to display messages in a conversation interface with different styles for sent and received messages.

Basic Chat Bubbles

Basic chat bubbles for sent and received messages.

Hello there! How are you doing today? I'm doing great, thanks for asking! How about you? 
<chat-bubble type="received">
    Hello there! How are you doing today?
</chat-bubble>

<chat-bubble type="sent">
    I'm doing great, thanks for asking! How about you?
</chat-bubble>

Chat Bubbles with Timestamp and Status

Chat bubbles showing timestamps and message status indicators.

Good morning! Did you get my email yesterday? Yes, I just saw it. I'll respond soon. Just sent you my response. Let me know if you need any clarification. 
<chat-bubble 
    type="received" 
    timestamp="2023-08-15T09:30:00">
    Good morning! Did you get my email yesterday?
</chat-bubble>

<chat-bubble 
    type="sent" 
    timestamp="2023-08-15T09:32:00"
    status="sent">
    Yes, I just saw it. I'll respond soon.
</chat-bubble>

<chat-bubble 
    type="sent" 
    timestamp="2023-08-15T09:35:00"
    status="delivered">
    Just sent you my response.
</chat-bubble>

<chat-bubble 
    type="sent" 
    timestamp="2023-08-15T09:40:00"
    status="read">
    Let me know if you need any clarification.
</chat-bubble>

Grouped Chat Bubbles

Consecutive messages from the same sender can be grouped together for a cleaner appearance.

Hey, I've been thinking about our project. I think we should focus more on the UI/UX aspects. What do you think? I completely agree. The user experience is critical. Let's schedule a design review meeting. 
<chat-bubble 
    type="received" 
    sender="John Doe"
    timestamp="2023-08-15T10:15:00">
    Hey, I've been thinking about our project.
</chat-bubble>

<chat-bubble 
    type="received" 
    sender="John Doe"
    grouped
    timestamp="2023-08-15T10:15:30">
    I think we should focus more on the UI/UX aspects.
</chat-bubble>

<chat-bubble 
    type="received" 
    sender="John Doe"
    grouped
    timestamp="2023-08-15T10:16:00">
    What do you think?
</chat-bubble>

Chat Bubbles with Sender

Received messages can display the sender's name.

Hello everyone! I'm looking forward to our meeting. Same here! I've prepared some notes to discuss. Great! I'll be sharing my screen to show the latest designs. 
<chat-bubble 
    type="received" 
    sender="John Doe"
    timestamp="2023-08-15T11:00:00">
    Hello everyone! I'm looking forward to our meeting.
</chat-bubble>

<chat-bubble 
    type="received" 
    sender="Jane Smith"
    timestamp="2023-08-15T11:02:00">
    Same here! I've prepared some notes to discuss.
</chat-bubble>

Custom Styled Chat Bubbles

Chat bubbles can be customized with different colors and styles.

This is a custom styled received message. And this is a custom styled sent message. 
<chat-bubble 
    type="received" 
    received-class="bg-purple-100 text-purple-900"
    timestamp="2023-08-15T12:00:00">
    This is a custom styled received message.
</chat-bubble>

<chat-bubble 
    type="sent" 
    sent-class="bg-green-100 text-green-900"
    timestamp="2023-08-15T12:02:00"
    status="read">
    And this is a custom styled sent message.
</chat-bubble>

API Reference

Chat Bubble Properties

Property Type Default Description
type string 'received' Type of message: 'sent' or 'received'
timestamp string undefined ISO datetime string or formatted time string
status string undefined Message status: 'sent', 'delivered', or 'read'
sender string undefined Name of the message sender (displayed for received messages)
grouped boolean false Whether this message is part of a consecutive group from same sender
sent-class string 'bg-blue-100 text-blue-900' Custom CSS classes for sent messages
received-class string 'bg-gray-100 text-gray-900' Custom CSS classes for received messages
class string '' Additional CSS classes for the chat bubble container
